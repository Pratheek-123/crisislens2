# CrisisLens 🎯
AI-Powered Rapid Crisis Response System

Built with Google Gemini API for GDG Build with AI (#BwAI)

## Problem
Delayed disaster response costs lives. Manual coordination 
fails during critical first minutes of a crisis.

## Solution
CrisisLens uses Gemini AI to triage disasters, dispatch 
resources optimally, and give coordinators a live ops view.

## Features
- AI Triage Engine (Gemini 2.0 Flash)
- Smart Resource Dispatch
- Live Crisis Map
- Real-time Incident Dashboard
- Event Log & KPI Tracking

## SDG Alignment
- SDG 3 — Good Health & Well-being
- SDG 11 — Sustainable Cities
- SDG 13 — Climate Action

## How to Run
1. Download index.html
2. Open in any browser
3. Paste your Gemini API key in the yellow bar
4. Go to AI Triage → select a scenario → Run

## Tech Stack
- Frontend: HTML, CSS, Vanilla JS
- AI: Google Gemini 2.0 Flash API
- Charts: Chart.js
- Maps: Canvas 2D API
